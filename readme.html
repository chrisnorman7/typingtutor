Welcome to Typing Tutor

Written by Chris Norman, the creator of Guitar By Touch.

This program aims to be simple and fun, and allows you to practise your typing.Exercises are simply written as text files, requiring you to type each letter correctly before moving on.

You are timed for each exercise, and your time, as well as how many letters you mistyped are shown at the end.

If you would like to make your own exercises, simply create them as text files, and (for ease of use), put them in the exercises sub directory of the main program folder.

Shortcut Keys

Typing Tutor includes a couple of shortcut keys, which are documented in the following table:






Key Description
Tab Repeats the current letter
F11 Toggles the playing of sounds
F12 Toggles the speaking of key hints
Control + F11 Turns the volume down by 10%
Control + F12 Turns the volume up by 10 %
